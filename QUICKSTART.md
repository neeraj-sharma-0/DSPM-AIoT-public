# Quickstart (Local or Colab)

Run notebooks in order under `notebooks/`.

Outputs are written to `outputs/`:
- `aiot_telemetry.csv`
- `discover_inventory.json`
- `classified_telemetry.parquet`
- `monitoring_summary.json`
- `governance_plan.parquet`
- `destroy_plan.csv` (simulated)
- `tombstones.jsonl` (simulated)
- `outputs/audit_logs/*.json`
